## Panasonic AK-HRP1000

Change camera selection on the AK-HRP1000/AK-HRP1005.

Note: This module does not poll or maintain an active connection to the unit, and the unit always returns an error even when it successfully receives and actions a command, thus the module does not report useful status updates.

**Minimum supported firmware:** `4.80-00-0.03`

### Setup

1. Go to menu item `AW CONTROLLER LINK`
2. Set the `RECEIVE PORT` and execute `INFO UPLOAD`
3. Set `AW CONT LINK` to `ON`

### Actions

- Select Camera
